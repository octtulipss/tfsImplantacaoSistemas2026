## Como Executar

### Pré-requisitos
- Sistema Linux (Ubuntu 20.04+ ou similar)
- Acesso sudo

### Instalação
Siga os comandos abaixo para realizar o deploy automatizado:

```bash
# 1. Copie o repositório
git clone https://github.com/octtulipss/tfsImplantacaoSistemas2026.git

# 2. Entre na pasta do repositório
cd TF01

# 3. Dê permissão de execução ao script de instalação
chmod +x scripts/install.sh

# 4. Execute o script de instalação
cd scripts

./install.sh
